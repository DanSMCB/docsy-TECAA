A informação nesta secção ajuda o utilizador a experimentar o teu projeto por conta própria.

- O que é que os utilizadores precisam de fazer para começar a usar o teu projeto? Isto pode incluir
  instruções de download/instalação, bem como quaisquer pré-requisitos ou
  requisitos de sistema.

- Um exemplo introdutório “Hello World”, se for apropriado. Tutoriais mais
  complexos devem ficar na secção de Tutoriais.

Considera utilizar os títulos abaixo para a tua página de primeiros passos.
Podes apagar qualquer um que não se aplique ao teu projeto.

## Pré-requisitos

Existem requisitos de sistema para utilizar o teu projeto? Que linguagens são
suportadas (se existirem)? Os utilizadores precisam de já ter algum software
ou ferramentas instaladas?

## Instalação

Onde pode o utilizador encontrar o código do teu projeto? Como o pode instalar
(binários, pacote instalável, compilação a partir do código-fonte)? Existem
várias opções/versões que podem ser instaladas e como devem escolher a mais
adequada?

## Configuração

Existe alguma configuração inicial que os utilizadores precisam de fazer após
a instalação para experimentar o teu projeto?

## Experimenta!

Os utilizadores podem testar a instalação, por exemplo executando um comando
ou implementando um exemplo “Hello World”?
