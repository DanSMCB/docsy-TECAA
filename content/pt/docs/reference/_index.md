Se o teu projeto tiver uma API, configuração ou outra referência — qualquer coisa que os utilizadores precisem de consultar a um nível ainda mais detalhado do que uma única tarefa — coloca-a (ou faz ligação para ela) aqui. Podes disponibilizar e ligar a documentação de referência gerada usando ferramentas como Doxygen, Javadoc ou outras, colocando os ficheiros na tua pasta `static/`. Descobre mais em [Adicionar conteúdo estático](https://docsy.dev/docs/adding-content/content/#adding-static-content).  

Para referência OpenAPI, o Docsy também fornece um [layout e shortcode Swagger UI](https://www.docsy.dev/docs/adding-content/shortcodes/#swaggerui) que apresenta o [Swagger UI](https://swagger.io/tools/swagger-ui/) usando qualquer ficheiro YAML ou JSON OpenAPI como fonte.
