Os tutoriais são **exemplos completos** compostos por **múltiplas tarefas** que
guiam o utilizador através de um cenário relativamente simples, mas realista:
por exemplo, construir uma aplicação que utilize algumas das funcionalidades do
teu projeto. Se já criaste alguns Exemplos para o teu projeto, podes basear
os tutoriais nesses exemplos.

Esta secção é **opcional**. No entanto, lembra-te que, embora possas não precisar
desta secção inicialmente, ter tutoriais pode ser útil para ajudar os teus
utilizadores a interagir com o código de exemplo, especialmente se houver aspetos
que necessitem de mais explicação do que é fácil fornecer através de comentários
no código.
